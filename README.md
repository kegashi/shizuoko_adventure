# スーパーしずオ

オリジナルキャラクター「しずオコ」くんを主役にした、ブラウザで動く2Dアクションゲームです。

## 起動方法

`index.html` をブラウザで開いてください。ローカルサーバーなしで動作します。

## 操作

- 自動で右へ進みます
- ジャンプ: `Space` / `↑` / `W`
- 傘攻撃: `X` / `K` / `Shift`
- スマートフォン: 画面下のボタン

## 目的

いちごを集めながら敵を傘で倒し、ステージ右端の家まで下校するとクリアです。敵に当たったり落下したり、時間が経つとオコ(体力)が減ります。オコが0%になると失敗です。

距離は、しずオコくんが1歩進む距離を1mとして表示します。リザルト画面にも到達距離が出ます。

敵には、通常巡回・高速移動・ジャンプ移動・浮遊・突進のバリエーションがあります。

## 攻撃

傘攻撃は、空振りしたときだけ少しオコを消費します。敵を倒したときはオコを消費せず、いちご2個分のスコアが入り、少しオコが戻ります。
